대학시절 MIS강의 과제로 만들었던 스크래치 게임의 오마주입니다.
벌레를 피하여 당근만 클릭하면 성공하는 게임으로,
매 스테이지 성공 시 난이도가 더 업그레이드 됩니다.

ES6문법을 사용하여 클래스를 이용하여 게임을 만들었고,
Object freeze로 상수관리, builder pattern을 이용하여 클래스 객체를 생성했습니다.
